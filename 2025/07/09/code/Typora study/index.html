1.标题：

​ 一级标题：Ctrl+1或# （ 注意：使用#号时不能缩进，并在#后添加空格后再写标题 ）

​ 二级标题：Ctrl+2或##

​ 三级标题：Ctrl+3或###

​ ……

​ 六级标题：Ctrl+6或######

2.文字：

​ 加粗：Ctrl+B

​ （在博客显示需要再** +加粗+** 两侧添加空格可正常显示）

​ 下划线：Ctrl+U

​ ==高亮==：内容两侧加上==

​ 斜体：Ctrl+i

​ 删除线：alt+shift+5

3.上下标：

​ 上标：内容两侧加上^ 如面积：m2 、体积：

​ 下标：内容两侧加上~ 如水：H2O 、双氧水：H2O2

​ (这里在博客显示有问题，hexo需要安装插件)

4.表格：

​ 法1：使用 | 来分隔单元格形成列，使用 - 来分隔表头和其他行形成行

( 注意：在用 - 时也要用 | 来分隔，并且在普通模式下，可能使用回车换行时会产生多余的空行，可以使用源代码模式查看是否有多余的空行并删除即可 )

​ 法2（推荐）：Ctrl+T 可以直接选择行数列数

5.列表：

5.1 无序列表：

​ Ctrl+shift+] 或 - +空格

  • 无序列表

5.2 有序列表：

​ Ctrl+shift+[ 或 序号数字. +空格

  2. 有序列表2

6.引用：

​ 一级引用：> 或 Ctrl+shift+Q

​ 二级引用：>>

​ ……

​ (注意：引用符号放在一行开头，且使用 > 时，需要在后面添加1个空格 )

二级引用

7.代码：

7.1 插入不确定代码：

​ Ctrl+shift+K

(注意：若使用搜狗输入法，Ctrl+shift+K是软键盘快捷键，两者会冲突，需要关闭 )

7.2插入确定代码：

​ 语言名称，如Python代码块: ```python + 回车

7.4 单个字符：

​ 内容两侧加上` ，内容会有矩形阴影，单独显示中间字符，如下所示：

python java

7.5 源代码模式：

​ Ctrl+/ 进行原浏览模式与源代码模式的互相切换

8.分割线：

​ — + 回车 会形成一条分割线

9.跳转：

9.1 外部跳转：超链接

​ [ 提示文字 ] + ( 网址 ) （注意 ：+号不需要打，这里只是方便浏览，后同 ）

​ 或 Ctrl+K

​ 百度

9.2 内部跳转：本文件内跳转

​ [ 提示文字 ] + ( #标题)

​ 或 Ctrl+K

​ 跳转到标题

10.自动链接：

​ URL或邮箱地址两侧添加< >

​ <www.baidu.com>

11.图片：

​ ! + [ 提示文字 ] + (图片地址)

​ 或 ! + Ctrl+K

12.数学公式：

12.1 希腊字母：

​ \希腊字母对应英文两侧添加$ （与LaTeX代码编写相同）

​ ，

12.2 打开数学公式编辑器：

​ Ctrl+shifit+M

(注意：若使用搜狗输入法，Ctrl+shift+M是系统菜单快捷键，两者会冲突，需要关闭 )

​ 数学公式换行：两个斜杠 \ 或 空格 + 一个斜杠 \ +空格

(这里在博客显示问题不会换行) ​ 数学公式、矩阵等与LaTeX代码编写相同

13.表情符：
